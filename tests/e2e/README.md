# End-To-End Test Migration Target

This directory is the target for browser and deployment-path smoke tests once e2e automation is introduced. It is README-only during Milestone 1.

Near-term migration purpose:

- hold browser acceptance automation for home, collection search, exam search, coverage UI, public artifact URLs, PWA/cache behavior, mobile viewport, and desktop viewport;
- provide evidence for milestones that require browser validation.

Manual browser acceptance remains required until automated e2e coverage exists and is wired into CI.
